SVG

SVG with glyph outlines provided by opentype.js, typesetting metrics by harfbuzz. Outlines are implemented as SVG symbols accessible via external resources, which is why we must

  • use HTML <object> tags instead of <img> tags (images are not allowed to reference external resources);
  • use a web server and not open the current file directly in the browser, as it would then not resolve the references in the loaded SVG.
